    columns-width

    @mixin columns-width($property, $columns, $position) { ... }

    Description

    Set the columns width for the provided property based on the column count and the current column position

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $property

    CSS property to apply the width

    		String— none
    $columns

    Column count

    		Number— none
    $position

    Current column position

    		Number— none

    Example

    Usage

    .foo {
  @include columns-width('max-width', 12, 4);
}

    CSS output

    .foo {
  max-width: 33.33333%;
}

    Requires

    • [function] get-width
    • [function] is-percent-unit

    Used by

    • [mixin] columns

    helper

    @mixin helper($selector, $bps: $breakpoints, $modifier: $modifier-delimiter) { ... }

    Description

    Generates the CSS code for the grid helpers including the code in the media queries using the projetc breakpoints

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $selector

    Helper classname selector

    		String— none
    $bps— noneList$breakpoints
    $modifier— noneString$modifier-delimiter

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    Usage

    @include helper('foo');

    CSS output

    .foo-xs {
  property: value;
}
@media screen and (min-width: 48em) {
  .foo-sm {
    property: value;
  }
}

    Requires

    • [mixin] queries
    • [variable] queries
